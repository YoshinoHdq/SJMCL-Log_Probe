## 简介

Log Probe 是一款 SJMCL 扩展，自动扫描 Minecraft 崩溃日志和错误报告，识别常见问题根因并给出解决建议。

使用官方脚手架重构，内置中英文双语切换。

## 功能特色

- 📄 自动扫描 Minecraft 错误日志与崩溃报告
- 🔍 识别 40+ 常见错误模式（内存不足、模组冲突、渲染错误等）
- 💡 为每种错误提供针对性的修复建议
- 🌐 设置面板一键切换中英文，无需安装不同语言版本
- 📋 支持导入日志文件或直接粘贴分析
- 🏠 首页卡片快捷查看最近分析结果

## 使用方法

1. 在 Releases 下载 .sjmclx 文件
2. 打开 SJMCL → 设置 → 扩展管理 → 导入
3. 在启动器首页卡片或扩展设置页中使用

## 兼容性

- 最低启动器版本: 1.2.0
