## 简介

Log Probe 是一款 SJMCL 扩展，自动扫描 Minecraft 崩溃日志和错误报告，识别常见问题根因并给出解决建议。

## 功能特色

- 📄 自动扫描 Minecraft 错误日志与崩溃报告
- 🔍 识别常见错误模式（内存不足、模组冲突、JDK 版本等）
- 💡 为每种错误提供针对性的解决建议
- 📋 支持查看完整日志原文

## 使用方法

1. 在 Releases 下载 org.yoshino.log_probe.sjmclx
2. 打开 SJMCL → 设置 → 扩展管理 → 导入
3. 启动器检测到崩溃或错误时自动弹窗分析，也可在扩展设置页中粘贴日志内容手动分析

> ⚠️ 启动器 1.1.3 版本暂不支持直接从错误日志窗口分析，如需分析请复制日志内容到扩展设置页中粘贴分析。

## 兼容性

- 最低启动器版本: 1.1.3
